# Security Policy

**Please do not open a public issue for security problems.**

If you find a vulnerability in any `stackbox-dev` repository, report it privately:

- Email **security@stackbox.xyz**, or
- If you're unsure, message a **dev-lead** directly.

Include what you found, how to reproduce it, and the affected repo/branch. We'll
acknowledge the report and follow up with next steps.

Do not disclose the issue publicly until we've confirmed it's resolved.
